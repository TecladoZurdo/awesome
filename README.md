## Configuración
- Luego proceder con la configuración

`````
mkdir ~/.config/awesome && cp /etc/xdg/awesome/rc.lua ~/.config/awesome/
`````

De lo contrario  clonamos este repositorio dentro de la carpeta
````
~/.config
````
